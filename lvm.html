Create and use a partition
Create a playbook called partition.yml that runs on all managed nodes that does the following:
Creates a single primary partition number 1 of size 1500MiB on device vdb
Formats the partition with the ext4 filesystem
Mounts the filesystem persistently at /mount1
-- If the requested partition size cannot be created, the error message "Requested size is not present" should be displayed and the size 800MiB should be used instead.
If the device vda does not exist, the error message "Devices is not present" should be displayed.
