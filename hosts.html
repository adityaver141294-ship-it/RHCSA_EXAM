Generate a hosts file
- Download an initial template file called hosts.j2 from http://classroom.example.com/content/EXAM/
to /home/admin/ansible/ Complete the template so that it can be used to generate a file with a line for each inventory host in the same format as /etc/hosts
Create a playbook called gen_hosts.yml that uses this template to generate the file /etc/hosts on hosts in the dev host group.
When completed, the file /etc/hosts on hosts in the dev host group should have a line for each managed host:
127.0.0.1 localhost localhost.localdomain localhost4 localhost4.localdomain4
::1 localhost localhost.localdomain localhost6 localhost6.localdomain6
172.25.250.10 servera.lab.example.com servera

172.25.250.12 serverc.lab.example.com serverc
