• CreateStratis file system:
  • Create 2 physical partitions in /dev/vdd with the size of 1GiB each.
  • Create stratis pool named as datastorage using /dev/vdd1 and /dev/vdd2 partitions
  • Create Stratis FileSystem named as data
  • Mount data filesystem on /storage directory
  • Stratis Filesystem will automaticaly mount at boot time by systemd
